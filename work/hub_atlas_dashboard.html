Notes — DRR (daily rental revenue per battery, USD) is computed as SUM(rental_income ÷ daily FX rate) ÷ SUM(available batteries that day) over the selected window. Settlement attributes (population, density, nightlight, households) come from MOPO's internal Atlas dataset, joined to each hub via point-in-polygon match. Nightlight uses Atlas sub-region resolution where available (172/426 hubs, mostly urban CD — Kinshasa, Kananga, Kikwit, plus Lagos/Abeokuta/Freetown clusters); falls back to parent settlement otherwise. Households is Atlas's count of satellite-detected building footprints in the polygon — independent of any census-derived population estimate. DRR computed as: SUM(rental_income / daily FX rate) ÷ SUM(available batteries that day). USD-denominated throughout. Lagos is included in this analysis (unlike SmartPlay analysis where it's excluded). Data pulled . DRR windows count back from this date.
